# 🎓 React Essentials Assignment - Student Grade Tracker

## 📖 Overview

This project was built as part of my React learning journey to understand **Class-Based Components** and **React Lifecycle Methods**. The application demonstrates state management, lifecycle methods, event handling, controlled forms, and component communication through props by building an interactive Student Grade Tracker.

---


## ✨ Features

- Add new students
- Remove students from the list
- Display student names, grades, and Pass/Fail status
- Grade validation (0–100)
- Highlight passed students in **Green**
- Highlight failed students in **Red**
- Filter and sort students
- Responsive and user-friendly interface

---

## ⚛️ React Concepts Used

- Class-Based Components
- State Management (`this.state`)
- `setState()`
- Props
- Event Handling
- Controlled Forms
- Conditional Rendering
- Lifecycle Methods:
  - `componentDidMount()`
  - `componentDidUpdate()`
  - `componentWillUnmount()`

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/react-essentials-assignment.git
```

Navigate to the project:

```bash
cd react-essentials-assignment/StudentGradeTracker
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Open **http://localhost:3000** in your browser.
