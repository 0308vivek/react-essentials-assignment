# React Essentials Assignment

This repository contains projects built while learning React fundamentals, covering component-based architecture, state management, React Hooks, Class Components, Lifecycle Methods, Context API, reducers, API integration, and modern UI development. Each project focuses on applying core React concepts through practical, hands-on applications.

---

## Projects

### 1. Portfolio Card Application

An interactive portfolio card built with React that focuses on reusable components, props, event handling, and state management.

#### Features

- Component-based architecture
- Dynamic skill badges using props
- Theme switching (Light/Dark Mode)
- Profile photo cycling
- Like counter using React state
- Interactive buttons and alerts

**Live Demo:** https://portfolio-card-kqpd.onrender.com/

---

### 2. Movie Database

A React-based movie database application focused on search functionality, dynamic filtering, conditional rendering, and favourite movie management.

#### Features

- Search movies using React state
- Dynamic movie filtering
- Real-time search results
- Conditional rendering for different search states
- Favourite/Unfavourite movie functionality
- Dedicated Favourite Movies section
- State-driven UI updates

**Live Demo:** https://movie-database-mi56.onrender.com/

---

### 3. Student Grade Tracker

A Student Grade Tracker built using **React Class-Based Components** to demonstrate state management, lifecycle methods, event handling, controlled forms, and component communication through props.

#### Features

- Add and remove students
- Display student grades and Pass/Fail status
- Grade validation (0–100)
- Filter and sort students
- Dynamic UI updates using `setState()`
- Lifecycle methods:
  - `componentDidMount()`
  - `componentDidUpdate()`
  - `componentWillUnmount()`
- Responsive interface

**Live Demo:** https://your-student-grade-tracker-link.onrender.com/

---

### 4. Task Manager Application

A Task Manager application built using React Context API and `useReducer` for centralized state management. The application allows users to create, edit, delete, and manage tasks while demonstrating scalable React architecture.

#### Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as complete/incomplete
- Clear all tasks
- Global state management using Context API
- State updates handled with `useReducer`
- Responsive design
- Smooth animations and transitions
- Modern task management UI

**Live Demo:** https://task-manager-vy9r.onrender.com/

---

### 5. Weather Dashboard

A responsive weather application built using React that demonstrates API integration and side-effect management with the `useEffect` hook. The application fetches real-time weather information based on user input and displays it with a clean, modern interface.

#### Features

- Search weather by city
- Live weather data using OpenWeatherMap API
- API fetching with `useEffect`
- Loading and error state handling
- Temperature, humidity, wind speed, and weather condition display
- Responsive and modern UI
- Controlled components using React state
- Side-effect management with React Hooks

**Live Demo:** https://weather-dashboard-49ht.onrender.com/
