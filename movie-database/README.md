# Movie Database

A React-based movie database application built as part of my React learning journey. This project focuses on implementing search functionality, filtering logic, conditional rendering, and favourite movie management using React state.

## Live Demo

https://movie-database-mi56.onrender.com

## Features

### Search Functionality

* Interactive search bar connected to component state.
* Real-time filtering of movie data.
* Automatic UI updates as the user types.

### Movie Filtering

* Utilizes local movie data.
* Dynamically filters movies based on search queries.
* Displays matching results instantly.

### Conditional Rendering

Displays different UI states for:

* No search input
* No matching movies found
* Successful search results

### Favourite Movies

* Mark movies as favourites.
* Remove movies from favourites.
* Maintain favourite status using React state.
* Display favourites in a dedicated section.
